Green Haven is a plant-centered full-stack application developed as part of a Master 1 academic project. The
project addresses the need for a digital platform that helps plant lovers identify plants, learn about plant care,
organize their personal plants, communicate with other users, and access a marketplace for plant-related
products. The system is composed of a mobile application, a desktop administration panel, and a backend
server connected to a cloud database.
The mobile application allows users to identify plants from images, save plants in a virtual garden, share
posts, use live chat, browse products, and place orders. Sellers can manage their products and follow
customer orders, while administrators can supervise users, posts, products, and general platform activity
through the admin panel. The mobile application was developed using React Native and Expo, the admin
panel using Electron, React, and Vite, and the backend using Node.js, Express.js, and MongoDB.
Several external services were integrated to improve the functionality of the system. Plant.id was used for
plant recognition, Google Gemini was integrated as a plant-focused AI assistant, Stream Chat was used for
real-time communication, and Cloudinary was used for online image storage. MongoDB Atlas was used to
host the database in the cloud, while the backend server was deployed on Render. In addition, an APK
version of the mobile application was generated to allow installation and testing on Android devices.
The development process included requirement analysis, feature planning, interface design, backend API
development, database modeling, authentication and authorization, external API integration, testing, and
deployment. Overall, Green Haven demonstrates the practical implementation of a complete full-stack
system that combines mobile development, backend services, database management, cloud hosting, artificial
intelligence integration, image handling, and marketplace features in one coherent academic project.
